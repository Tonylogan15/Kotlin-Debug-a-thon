Submitted by: Antonio Logan

Time spent: 3 hours spent in total

Summary

Kotlin Debug-a-thon is an Android app that initially contained several intentional Kotlin bugs spread across different activities. After fixing them, the app now correctly performs navigation, displays calculated values, shows the current day, generates random colors, prints lists, and displays a favorite number. This project strengthened my debugging skills, improved my understanding of Kotlin error handling, and gave me more confidence navigating Android Studio.
